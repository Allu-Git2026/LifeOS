# LifeOS

LifeOS is a full-stack productivity platform built using React, Node.js, Express, PostgreSQL, Prisma, and Tailwind CSS.

## Features

- Authentication (JWT)
- Tasks Management
- Habit Tracking
- Notes
- Goals Progress
- Productivity Dashboard
- Analytics

## Tech Stack

Frontend:

- React
- TypeScript
- Tailwind CSS
- Axios

Backend:

- Node.js
- Express
- Prisma

Database:

- PostgreSQL

## Run

Frontend:

```bash
npm run dev
```

Backend:

```bash
npm run dev
```
